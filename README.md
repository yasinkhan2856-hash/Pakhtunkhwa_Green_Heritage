# 🌿 KPK Plant Explorer

A Flutter mobile application that provides structured information about plant species found in **Khyber Pakhtunkhwa (KPK), Pakistan**.
This app is designed for **students, researchers, teachers, and plant enthusiasts** to explore regional flora in an easy and interactive way.

---

## 📱 Features

* 🏠 Clean Home Screen with navigation cards
* 🌿 Category-wise plant browsing system
* 📄 Clickable plant list (each plant opens detail page)
* 🧾 Detailed plant information (image, name, family, habitat, uses)
* 🗺️ Interactive Plant Map of KPK
* 🎨 Nature-themed UI (green & white design)
* ⚡ Fast and lightweight Flutter app

---

## 📂 Plant Categories

* Flora of KPK
* Mixed Annual Plants
* Indigenous Plants
* Weeds
* Invasive Species
* Medicinal Plants
* Endangered Plants
* Plant Map of KPK

---

## 🌱 App Workflow

1. Open the app
2. Select a plant category
3. View list of plants
4. Tap on any plant name
5. See full plant details page

---

## 📖 Plant Detail Includes

* Plant Image
* Common Name
* Botanical Name
* Plant Family
* Habitat / Location
* Description
* Uses / Environmental Impact

---

## 🗺️ Map Feature

* Interactive map of KPK region
* Plant location markers
* Tap marker to view plant info
* Navigate to full plant detail page

---

## 🛠️ Tech Stack

* Flutter (Frontend)
* Dart (Logic)
* Firebase / JSON (Database)
* Google Maps / OpenStreetMap (Map Integration)
